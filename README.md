# Foodie - MERN Food Ordering Platform

## Features

* User Authentication (JWT)
* Food Menu Management
* Shopping Cart
* Order History
* Admin Dashboard
* CRUD Operations
* MongoDB Atlas Integration

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Render
* Vercel
